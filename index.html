GeoSuite Help

GeoRef Mode
1. Upload a raster image (scanned map, aerial photo, etc.).
2. Click on known locations to place Ground Control Points (GCPs).
3. Enter geographic coordinates (lon, lat) for each GCP, or use Quick Corner Bounds.
4. Export as GeoTIFF, World File, or send directly to Digitizer mode.

Digitizer Mode
1. Upload GeoTIFFs (auto-positioned by georeference metadata) or plain images.
2. Use the draw toolbar to create vector features (polygons, lines, markers, rectangles, circles).
3. Click features to edit names. Use Measure Distance/Area tools for quick measurements.
4. Import/export GeoJSON from the Data tab.

Integration
Use "Send to Digitizer" in GeoRef export to overlay your georeferenced image on the Leaflet map for digitizing.
